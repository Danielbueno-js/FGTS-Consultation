# FGTS-Consultation

Consulta para retirada de saque aniversario do seu FGTS

Pré requisitos

- npm e Node.js 20 +

Para executar o Projeto primeiro 

- git clone <url-do-repositorio>
- cd fgts-consultation


então baixe as dependencias

- npm install

e então finalmente execute o projeto

 - npm run dev
